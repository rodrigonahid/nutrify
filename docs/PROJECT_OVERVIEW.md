# Nutrify - Nutrition Management App

## Overview
Nutrify is a nutrition management platform that connects nutritionists with their patients, allowing for efficient patient management and tracking.

## Target Users
- **Admins**: System administrators who manage the platform
- **Nutritionists**: Healthcare professionals who manage patients
- **Patients**: End users receiving nutritional guidance

## Core Features

### Phase 1: Authentication & User Management (Completed)
- Multi-role authentication system (Admin, Nutritionist, Patient)
- Admin user creation via CLI
- Nutritionist user creation via CLI script
- Admin dashboard for nutritionist management
- Patient onboarding via invite codes
- Role-based access control

### Phase 2: Progress Tracking (Current)
- Detailed body composition tracking
- Perimeter measurements (trunk, arms, legs)
- Skinfold measurements (9 points)
- Professional recording interface
- Patient progress viewing
- Historical comparison

### Future Phases
- Patient nutrition plans
- Meal tracking
- Analytics and reporting
- Communication tools

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Database**: PostgreSQL
- **ORM**: Drizzle
- **UI**: shadcn/ui + Tailwind CSS
- **Language**: TypeScript
